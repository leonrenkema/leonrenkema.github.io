# Navigation

  * **^** Beginning of line
  * **$** End of line
  * **W** Next word
  * **E** Next word (end)
  * **{** Previous paragraph
  * **}** Next paragraph
  * **]]** Next function
  * **[[** Beginning this function
  * **H** Top screen
  * **L** bottom screen

# Substitute
  * Search and replace '':%s/{regx}/{replace}/g''

# File handling
  * **:ls** Show open files
  * **:bn** Open next file
  * **:bp** Open previous file
  * **ZZ** Save and quit

# File explorer
  * **:Explore** Open file explorer
  * **i** Cycle through views

# Window
  * **:sp**lit Horizontally split window
  * **:vsp**lit Vertically split window
  * **^w <up><down><right><left>** Activate window up,down,right,left of the current

# Settings
  * **set nu** Turn on line numbers
